Data formatting.

This visualization expects a properly formatted .csv file. It should have one date per row in "MM/DD/YYYY" format. The first row should be a header row with columns for "minutes" and "time." Similar formats might work, but we make no promises.

      minutes,time
      36,1/25/2010
      27,1/8/2010
      36,1/7/2010
      8,1/17/2010
      29,1/11/2010
      22,1/28/2010
      ...
